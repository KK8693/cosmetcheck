# RDC 529/2021 禁用成分翻译模板

## 文件说明
- **RDC529_translation_template.csv**: 翻译模板文件
- 列说明：
  - `num`: 序号 (1-1299)
  - `name_pt`: 葡萄牙语原名
  - `name_zh`: 中文翻译（留空，请填写）
  - `cas`: CAS号
  - `ec`: EC号
  - `notes`: 备注

## 翻译要求
1. 请在 `name_zh` 列填写中文翻译
2. 翻译原则：
   - 化学名称：使用标准中文化学名称
   - 植物名称：保留拉丁学名，添加中文
   - 括号内英文：保留不翻译
   - *号表示需要特别注意的成分

## 翻译完成后
请将文件保存回原路径，覆盖原文件即可。
